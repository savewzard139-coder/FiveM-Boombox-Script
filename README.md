# FiveM-Boombox-Script https://www.youtube.com/watch?v=lw2pNdhNEt0
Looking for an easy way to play music on your FiveM server? Try Camou_Boombox.  Main features:  Precise placement with preview (rotate with scroll) Custom playlists (no database needed) Supports YouTube, MP3, and radio (via xSound) Volume and range control Clean UI (ox_lib, ox_target)  Dependencies: ox_lib, ox_target, ox_inventory, xSound

Required
xSound

ox_lib

ox_inventory

ox_target

Install the necessary resources.

Edit the configuration as you wish.

Start the script in server.cfg.

And then have fun!
